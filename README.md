## 功能点

- 加载小车
  - 车架
  - 车轮
  - 刹车
- 加载按钮
  - 按钮位置
  - 按钮点击事件
  - 按下 刹车
  - 松开 恢复行驶
- 粒子背景

  - 行驶粒子倾斜 30° 向下缓动
  - 刹车粒子静止
